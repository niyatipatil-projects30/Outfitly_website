# E-Commerce Website
## Project Overview

This is a responsive E-Commerce Website developed using React and Vite. The project allows users to browse products and view product details through a clean and user-friendly interface.

## Features

* Responsive design
* Product listing page
* Product detail view
* React Router navigation
* Reusable React components

## Technologies Used

* React
* Vite
* JavaScript
* CSS
* React Router DOM
