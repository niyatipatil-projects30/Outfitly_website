# Outfitly

Outfitly is a fashion recommendation web application that helps users discover outfits based on their preferences, making it easier to explore stylish clothing combinations.

## ✨ Features

- 👕 Browse fashion collections
- 🎨 Clean and modern user interface
- 📱 Responsive design
- 🔍 Easy outfit exploration
- ⚡ Fast and interactive experience

## 🛠️ Tech Stack

### Frontend
- React
- Vite
- CSS

### Backend
- Node.js
- Express.js
- Supabase
